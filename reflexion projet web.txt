

PARTIE 1 : Les types de sites web 

blog: site web dynamique qui permet de rédiger des articles les modifier et pouvoir les supprimer généralement relier a un serveur pour stocker les articles
exemple : Le Journal le monde est un bon exemple

Portfolio: site web statique contenant les projets d'un développeur web généralement intégrée à un site vitrine qui permet de mettre en avant son travaille effectuer dans ça carrière.

E-commerce: site dynamique qui permet de consulter des articles de pouvoir utiliser un panier et ensuite passer à la page de paiement.
exemple: ldlc.com

Site vitrine: un site généralement statique qui abrite comme une devanture de magasin , il permet de présenter une entreprise ou une enseigne sans 
boutique en ligne, pour un dév généralement composer d'un Portfolio et (ou) CV.

exemple: leodevtech.fr mon site vitrine qui présente mes projets & mon CV.
